The process

Three steps, zero effort

Configure once, receive daily. Specola runs silently in your menubar and handles everything.

01 — FETCH

Concurrent RSS harvest

Specola parses your OPML file and fetches all feeds in parallel with 20 threads. Failed feeds are skipped gracefully — one broken source never blocks the rest.

02 — ANALYZE

Your AI reads everything

The full digest is sent to your chosen LLM — Claude Code CLI, OpenAI Codex CLI, or a local model via LM Studio. It prioritizes, connects dots, and writes analytical paragraphs — not one-line summaries.

03 — DELIVER

Your format, your way

Choose DOCX, PDF, or EPUB — Specola renders a professionally typeset report with clickable links, branded headers, and clear hierarchy. An HTML version is always generated for the browsable portal archive.

Not a summary.
An intelligence brief.

Each Specola is a multi-page briefing with deep analysis, source links, and actionable insights — delivered in your chosen format and archived in a browsable HTML portal.

Notification Center widget

Clickable hyperlinks to every original article in every format. The briefing is a starting point for deeper reading, not a dead end.

Under the hood

Three components, one contract

The Swift app handles UI and scheduling. The Python engine does the analysis and rendering. A WidgetKit extension shows highlights in Notification Center. All connected via JSON and App Groups.

Get started

Three commands to
your first briefing

# Clone and build git clone https://github.com/amargiovanni/specola.git cd specola xcodebuild -project Specola.xcodeproj -scheme Specola -configuration Release -derivedDataPath build # Install cp -R build/Build/Products/Release/Specola.app /Applications/

The Python engine is bundled inside the app and automatically set up on first launch — no manual Python configuration needed. Choose your LLM provider in Settings: Claude Code CLI (default), OpenAI Codex CLI, or LM Studio for local models. Import your OPML feeds, write your profile, and hit “Genera ora”. Your first briefing is minutes away.
