Sotto il cofano

Tre componenti, un contratto. L'app Swift gestisce UI e scheduling. Il motore Python fa analisi e rendering. Il widget WidgetKit mostra i punti chiave nel Centro Notifiche.

Tre comandi per il
tuo primo briefing

# Clona e compila git clone https://github.com/amargiovanni/specola.git cd specola xcodebuild -project Specola.xcodeproj -scheme Specola -configuration Release -derivedDataPath build # Installa cp -R build/Build/Products/Release/Specola.app /Applications/

Il motore Python è incluso nell'app e si configura automaticamente al primo avvio. Scegli il provider AI nelle impostazioni, importa i feed OPML, scrivi il tuo profilo e premi “Genera ora”.

Domande frequenti

Dipende dal provider. Con Claude Code CLI e LM Studio non serve. Con OpenAI sì, ma la configuri una sola volta nel setup guidato.
Sì, con LM Studio. Scarichi il modello una volta e Specola analizza tutto in locale, senza mai inviare dati fuori dal tuo Mac.
Specola è gratuito e open source (MIT). Paghi solo l'eventuale uso API del provider AI che scegli. Il pre-filtro riduce i costi del 60-75%.
Qualsiasi Mac con macOS 14 (Sonoma) o successivo. Apple Silicon raccomandato per LM Studio locale.
Sì. Importi un file OPML con le tue fonti RSS, oppure le aggiungi una alla volta. Puoi organizzarle per categoria.
DOCX, PDF ed EPUB. Ogni briefing è anche disponibile nel portale HTML integrato per la consultazione via browser.
